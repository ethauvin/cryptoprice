## Kotlin Example
To compile & run the Kotlin example:

```console
./bld compile

./bld run
./bld run --args="btc"
./bld run --args="eth eur"
```

## Java Example

To compile & run the Java example:

```console
./bld compile

./bld run-java
./bld run-java --args="btc"
./bld run-java --args="eth eur"
```
